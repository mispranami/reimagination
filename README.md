# Riot Nexus

A modern web application for exploring Riot Games data across League of Legends, Valorant, and TFT.

## Project Structure

- `/backend` - Node.js Express API server
- `/frontend` - Next.js 14 application with Tailwind CSS

## Getting Started

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
